# Example App

This is a minimal example of how to use `@ztachi007/react-native-drag-sort-strip`.

## Running the Example

1. Create a new Expo project (or use an existing one):

```bash
npx create-expo-app my-app
cd my-app
```

2. Install the library and dependencies:

```bash
npm install @ztachi007/react-native-drag-sort-strip react-native-reanimated react-native-gesture-handler
```

3. Copy the content of `App.tsx` from this folder into your project's `App.tsx` (or `app/index.tsx`).

4. Run the project:

```bash
npm run start
```

## Features Demonstrated

- **Basic Drag & Sort**: Long press to reorder items.
- **Drop to Delete**: Drag an item into the "Trash Zone" to delete it.
- **Dynamic Data**: Add new items to the list.
- **Custom Styling**: Fully customizable item rendering.
